Documentação

  • Descrição: A aplicação consiste em um sistema de gerenciamento de arquivos simples, com uma interface de usuário construída utilizando páginas HTML e estilizada com CSS para melhorar a experiência do usuário.
  • Servidor HTTP: Implementa um servidor HTTP em Python, utilizando a biblioteca "socket", que é capaz de interpretar os métodos HTTP GET e PUT para realizar operações de leitura e inserção de arquivos.
  • Funcionalidades: Os usuários podem interagir com a aplicação por meio de uma interface web intuitiva, que permite visualizar arquivos presentes no servidor e adicionar novos.
  • Operações suportadas:
    GET: Utilizado para recuperar os arquivos existentes no servidor.
    PUT: Permite a inserção de novos arquivos no servidor.
  • Benefícios: Essa abordagem oferece uma maneira eficiente e flexível de interagir com os dados armazenados no servidor, proporcionando uma experiência amigável e funcional para os usuários.
